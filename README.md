# Git Activity Dashboard

## Overview

Git Activity Dashboard is a developer productivity application built using Python, Flask, and Microsoft SQL Server.

The application provides a centralized view of development activity across local Git repositories, helping developers review recent work, monitor project activity, and generate summarized reports.

Designed as a learning-focused portfolio project, it combines backend development, database management, and Git integration into a practical developer tool.

---

## Features

### Repository Management

* Discover and register local Git repositories
* Organize project activity in a single dashboard

### Activity Tracking

* Monitor repository activity
* Review commit history
* Explore project-specific development records

### Commit Categorization

* Automatically group commits into meaningful categories
* Improve visibility into development patterns

### Repository Filtering

* View activity for individual repositories
* Navigate project history more efficiently

### Weekly Reporting

* Generate concise summaries of development activity
* Review project engagement over time
